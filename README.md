# Netflix Clone 🎬

A responsive **Netflix Landing Page Clone** built using **HTML5** and **CSS3**. This project recreates the look and feel of Netflix's homepage, focusing on modern UI design, responsive layouts, and clean code practices.

## 🌟 Features

* 🎥 Netflix-inspired landing page
* 📱 Fully responsive design
* 🎨 Modern and clean user interface
* 🖼️ Hero banner with background image
* 📺 Feature sections with images and text
* ❓ FAQ section
* 📧 Email signup section
* 🔻 Footer with useful links

## 🛠️ Technologies Used

* HTML5
* CSS3

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Chanchal-web/Netflix-Clone.git
```

### 2. Open the project

Navigate to the project folder and open the `index.html` file in your web browser.

No additional installation or dependencies are required.
